#### 安装教程

1.  ``` git clone https://github.com/lifaxin/fx-blog.git ```
2.  ``` mvn clean package -Dmaven.test.skip=true ```
3.  maven打包成功后，可将打好的包上传至服务器，通过 ``` java -jar *.jar ``` 启动
